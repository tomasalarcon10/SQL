# 🗄️ SQL Projects

Proyectos de analítica de datos y business intelligence resueltos con SQL, enfocados en casos de negocio reales: finanzas, retail, RRHH, energía, logística y geografía de clientes.

## 📂 Contenido

- **Customer_Lifetime_Value_Project** — Cálculo y análisis del valor de vida del cliente (CLV) para segmentación y estrategia comercial.
- **Global_Debt_Intelligence_System** — Análisis de deuda global a nivel país/región.
- **IBM_HR_Attrition_Intelligence** — Análisis de rotación de personal (attrition) usando el dataset de RRHH de IBM.
- **Olist_Customer_Geographic_Analytics** — Análisis geográfico de clientes usando el dataset de e-commerce brasileño Olist.
- **Retail_Revenue_Intelligence_Hub** — Análisis de ingresos y desempeño en el sector retail.
- **Ride_Hailing_Driver_Signup_Analytics** — Analítica de registro de conductores en plataformas de ride-hailing.
- **Sales_Analysis_System** — Sistema de análisis de ventas.
- **Smart_Electricity_Billing_Platform** — Análisis de facturación eléctrica inteligente.

## 🎯 Objetivo
Aplicar SQL para resolver problemas reales de negocio: consultas complejas, joins, agregaciones, ventanas analíticas (window functions) y modelado de KPIs.

## 🛠️ Tecnologías
SQL · Python (para carga/procesamiento de datos)
